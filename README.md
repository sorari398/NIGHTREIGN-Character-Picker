# ナイトレイン ランダム夜渡り (NIGHTREIGN-Character-Picker)

『ナイトレイン』のセッションやマルチプレイをより円滑に、そして盛り上げるためのファン制作ツールです。
プレイヤーキャラクター（夜渡り）と、討伐対象（標的）をランダムに決定し、作戦指示書のようなデザインの画像を生成・Discordへ共有できます。

## 📋 主な機能

* **夜渡りランダム選出**: 1〜3名のプレイヤーキャラクターをランダムにピックアップ。
* **標的（ボス）選出**: 討伐対象をランダムに決定。低確率（約20%）で強敵「常夜の王」が出現するギミック付き。
* **プレイヤー名反映**: 入力したプレイヤー名を画像内の各キャラクター上部に表示。
* **画像内ダイナミック注釈**: 「常夜の王」が出現した際のみ、特殊ルールを補足する注釈を画像内に自動表示。
* **Discord Webhook連携**: 生成された画像をボタンひとつでDiscordチャンネルへ直接投稿。

## 🚀 使い方

1.  **Discord連携（任意）**:
    Discordのチャンネル設定から取得した「Webhook URL」を貼り付けます。
2.  **プレイヤー名の入力**:
    参加するプレイヤーの名前を入力します。空欄の場合は自動的に「Player 1〜3」として処理されます。
3.  **オプション選択**:
    「キャラ重複の許可」や「標的の選出」を必要に応じてチェックします。
4.  **抽選開始**:
    「抽選開始 & 画像送信」をクリックすると、ブラウザ上にプレビュー画像が表示され、設定済みの場合はDiscordへ送信されます。
